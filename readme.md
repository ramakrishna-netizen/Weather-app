## API Reference

#### Get all items

```http
  GET https://api.openweathermap.org/data/2.5/weather?units=metric&q=id&api_key
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |



| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id as state name  |
